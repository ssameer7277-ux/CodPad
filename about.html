📚 HTML Tutorial for Beginners

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. Every website you visit on the internet is built using HTML. Learning HTML is the first step to becoming a web developer.

What is an HTML Tag?

HTML uses special codes called tags to tell the browser how to display content. A tag has an opening part and a closing part. For example:

<h1>This is a Heading</h1> <p>This is a paragraph of text.</p>

Basic HTML Page Structure

Every HTML page starts with the same basic structure. Here is what a simple HTML page looks like:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Web Page</title> </head> <body> <h1>Hello World!</h1> <p>Welcome to my website.</p> </body> </html>
💡 Tip: You can type this code in CodePad and see the result instantly in the live preview panel!

Common HTML Tags

Here are the most commonly used HTML tags that every beginner should know:

Heading Tags: Used to create titles and headings on your page.

<h1>Biggest Heading</h1> <h2>Second Heading</h2> <h3>Third Heading</h3>

Paragraph Tag: Used to write normal text content.

<p>This is a paragraph. Write your text here.</p>

Link Tag: Used to create clickable links.

<a href="https://example.com">Click Here</a>

Image Tag: Used to display images on your page.

<img src="photo.jpg" alt="My Photo">

List Tags: Used to create bullet point lists.

<ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul>

HTML Tags Quick Reference

Here are some important HTML tags:

<div> <span> <strong> <em> <br> <hr> <table> <form> <input> <button> <header> <footer> <nav> <section> <article>

🎨 CSS Styling Guide

CSS stands for Cascading Style Sheets. It is used to make your HTML pages look beautiful. With CSS, you can change colors, fonts, sizes, and layouts of your web page elements.

How to Add CSS

You can add CSS directly inside your HTML file using the style tag:

<style> body { background-color: #0e1117; color: white; font-family: Arial, sans-serif; } h1 { color: #6c63ff; font-size: 2em; } p { line-height: 1.7; opacity: 0.8; } </style>

CSS Colors

Colors in CSS can be written in different ways:

/* Color name */ color: red; /* Hex code */ color: #6c63ff; /* RGB */ color: rgb(108, 99, 255);

CSS Box Model

Every HTML element is a box. You can control the space inside and outside the box using padding and margin:

.box { padding: 20px; /* Space inside */ margin: 10px; /* Space outside */ border: 1px solid #ccc; border-radius: 8px; }
💡 Tip: Practice writing CSS in CodePad and see the changes instantly in the live preview!

⚡ JavaScript Basics

JavaScript is a programming language that makes your web pages interactive. With JavaScript, you can create buttons that respond to clicks, show or hide content, and much more.

Your First JavaScript

<button onclick="sayHello()">Click Me</button> <p id="msg"></p> <script> function sayHello() { document.getElementById('msg').textContent = 'Hello World!'; } </script>

JavaScript Variables

<script> let name = "CodePad"; let version = 1.0; let isFree = true; console.log(name); // CodePad console.log(version); // 1.0 </script>
